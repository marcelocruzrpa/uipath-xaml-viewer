Platform

GitHub Personal Access Token

A token increases the GitHub API rate limit from 60 to 5,000 requests per hour and is required for private repositories. The token is stored locally in this browser profile only and is not synced across devices.

- Manage tokens on GitHub

Enter the hostname of your GitHub instance (e.g., github.mycompany.com).

Registering a host enables the extension on that instance. A token is optional (only needed for private repos or higher rate limits).

GitLab Personal Access Token

A token is required for private repositories and increases API rate limits. Create a token with read_api scope. The token is stored locally only.

- Manage tokens on GitLab

Enter the hostname of your GitLab instance (e.g., gitlab.mycompany.com).

Registering a host enables the extension on that instance. A token is optional for public projects.
